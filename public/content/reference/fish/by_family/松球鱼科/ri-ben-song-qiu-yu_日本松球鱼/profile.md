# 日本松球鱼

- 来源：https://fishweather.cn/fish/ri-ben-song-qiu-yu
- 学名：Monocentris japonica
- 科：松球鱼科
- 纲：辐鳍鱼纲
- 目：金眼鲷目
- 水层：底层
- 食性：肉食性，主要以小型甲壳类动物如虾、蟹幼虫等为食，也捕食小型底栖无脊椎动物。

- 适温：22-28 °C

日本松球鱼
Monocentris japonica
别名：
菠萝鱼、松球鱼
基础档案
体呈卵圆形且侧扁，全身披有厚重骨质鳞片，形成坚硬外壳。头部比例较大，口小，吻部突出。体色鲜艳，呈亮黄色至橙黄色，每片鳞片边缘为黑色，如同菠萝或松球，故得名。腹鳍特化为一枚硬棘。

主要栖息于岩礁、珊瑚礁区域的洞穴或阴暗处，也见于港湾或潮间带水坑中。
