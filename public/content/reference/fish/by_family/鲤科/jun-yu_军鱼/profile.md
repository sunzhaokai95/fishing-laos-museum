# 军鱼

- 来源：https://fishweather.cn/fish/jun-yu
- 学名：Spinibarbus hollandi
- 科：鲤科
- 纲：辐鳍鱼纲
- 目：鲤形目
- 水层：底层,中层
- 食性：杂食性，主要摄食藻类、水生昆虫、小型底栖动物及植物碎屑
- 栖息环境：江河中上游多砾石或卵石的Rapids和潭涧中
- 适温：18-25 °C

别名：

军鱼
为国家保护动物
根据《中华人民共和国野生动物保护法》，
军鱼
属于保护物种， 禁止非法捕捞、买卖、食用。如在垂钓过程中误钓，请立即放生。 本页仅提供科普识别信息，不包含任何钓法或饵料建议。
基础档案
体长，圆筒形，稍侧扁。头锥形，吻圆钝。口下位，马蹄形，唇厚，具发达的乳突。须2对，吻须较粗长。眼小，侧上位。鳞小，侧线完全。体背部灰褐色，腹部银白色。各鳍灰黄色。
