# 花鳍燕鳐鱼

- 来源：https://fishweather.cn/fish/hua-qi-yan-yao-yu
- 学名：Cypselurus poecilopterus
- 科：飞鱼科
- 纲：辐鳍鱼纲
- 目：颌针鱼目
- 水层：表层
- 食性：主要以浮游生物为食，包括浮游甲壳类、小型鱼卵和幼鱼等。
- 栖息环境：热带和亚热带海洋的表层水域，通常远离海岸，属于大洋性鱼类。
- 适温：22-28 °C

花鳍燕鳐鱼
Cypselurus poecilopterus
别名：
飞鱼
基础档案
体延长，略侧扁，呈纺锤形。头中大，吻短钝。口上位，下颌略突出。眼大，侧位。胸鳍特别发达，向后延伸超过臀鳍起点，鳍条具分支，飞翔时展开如翼。腹鳍短小，位于腹部中央略后。背鳍和臀鳍相对，位于身体后部。尾鳍深叉形，下叶略长于上叶。体背部蓝灰色，腹部银白色。胸鳍浅灰色，具有明显的黑色斑点，故名斑鳍。
