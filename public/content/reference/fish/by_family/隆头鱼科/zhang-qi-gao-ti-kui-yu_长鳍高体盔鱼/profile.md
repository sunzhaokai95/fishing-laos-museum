# 长鳍高体盔鱼

- 来源：https://fishweather.cn/fish/zhang-qi-gao-ti-kui-yu
- 学名：Pteragogus aurigarius
- 科：隆头鱼科
- 纲：辐鳍鱼纲
- 目：鲈形目
- 水层：底层,近底层
- 食性：肉食性，主要以小型底栖无脊椎动物为食，如小型甲壳类、多毛类、软体动物等。也会捕食小型鱼类。

基础档案
金丝海龙体延长而侧扁，呈纺锤形。头部中等大小，吻尖。眼睛较大，位于头部上方。口小，端位。体被栉鳞，侧线完全。背鳍连续，具硬棘和软条；臀鳍短小；胸鳍圆形；尾鳍截形。体色多变，通常为黄褐色至红褐色，体侧有多条金色或白色纵纹，如同丝线，故名金丝海龙。各鳍颜色与体色相近，有时边缘略带浅色。
