# 裸狐鲣

- 来源：https://fishweather.cn/fish/luo-hu-jian
- 学名：Gymnosarda unicolor
- 科：鲭科
- 纲：辐鳍鱼纲
- 目：鲈形目
- 水层：中上层
- 食性：肉食性，主要捕食各种鱼类、乌贼、甲壳类等。幼鱼阶段主要摄食小型鱼类和浮游生物。
- 栖息环境：主要栖息于热带和亚热带的珊瑚礁区和近岸水域，也见于外海。常在水质清澈、富含氧气的水域活动。
- 适温：22-30 °C

基础档案
体延长，粗壮，呈纺锤形，横截面近圆形。头大，吻尖。眼小，脂眼睑不发达。口大，端位，斜裂。上下颌具强大圆锥形犬齿，故名犬齿金枪鱼。体被小圆鳞，胸部鳞片较大。第一背鳍长，具13-15鳍棘，第二背鳍和臀鳍后方各具7-9个小鳍。胸鳍短小。尾鳍深叉形。体背部蓝灰色或深灰色，腹部银白色，体侧无斑纹。

怎么钓
拖钓
拖钓犬齿金枪鱼通常使用活饵或大型拟饵，如大型米诺、深潜型颤泳饵、章鱼拟饵等。拖钓速度不宜过快，保持在犬齿金枪鱼的巡游速度范围内。当鱼咬钩时，竿尖会大幅度下弯，此时应立即收紧泄力，进行扬竿刺鱼，并与鱼展开搏斗。
路亚海钓
路亚海钓犬齿金枪鱼主要采用抛投或船钓方式，标点选择在珊瑚礁区、深海礁石、沉船残骸等犬齿金枪鱼常出没的区域。操控路亚时，可采用快速回收、抽动、跳跃等手法，模拟猎物逃窜的姿态，吸引犬齿金枪鱼的攻击。中鱼后，需利用竿身腰力与鱼搏斗，注意泄力调节，避免断线跑鱼。

实战技巧
选择合适的钓点:犬齿金枪鱼常在水流湍急、地形复杂的海域活动，如礁盘边缘、海沟附近等。
使用合适的装备:针对犬齿金枪鱼的力量和牙齿特点，选择强度足够的路亚竿、线组和拟饵。
掌握正确的操控手法:根据不同的拟饵类型和鱼情，采用不同的操控手法，使拟饵生动逼真。
搏鱼技巧:中鱼后，要保持冷静，合理泄力，与鱼周旋，避免硬碰硬导致断线跑鱼。
注意安全:海钓存在一定风险，务必穿着救生衣，注意天气变化和海况，确保安全。
