# 多棘腔吻鳕

- 来源：https://fishweather.cn/fish/duo-ji-qiang-wen-xue
- 学名：Coelorinchus multispinulosus
- 科：长尾鳕科
- 纲：辐鳍鱼纲
- 目：鳕形目
- 水层：深海底层
- 食性：主要以底栖无脊椎动物为食，如甲壳类、多毛类等。
- 栖息环境：栖息于深海大陆坡和海山，通常在水深200-800米处。
- 适温：5-15 °C

Coelorinchus multispinulosus
别名：
无
基础档案
体延长，侧扁，尾部细长。头大，吻尖突。眼大，圆形。口下位，横裂。颏须短小。体被栉鳞，粗糙。体背部灰褐色，腹部银白色。各鳍浅色。背鳍2个，第一背鳍棘状，第二背鳍与臀鳍相对。尾鳍尖细。

怎么钓
深海船钓
深海船钓主要采用沉底钓法或铁板钓法。将钓组或铁板沉至海底，通过船的慢速移动或人为抽动，模拟饵料的动态，吸引深海鱼类捕食。由于水深，需要使用重型铅坠或铁板，并借助电动卷线器收线。

实战技巧
精准定位钓点:利用鱼探仪等设备，准确找到鱼群聚集区或海底结构复杂区域，提高命中率。
合理搭配钓组:根据目标鱼种、水深、流速等因素，选择合适的钓竿、线组、铅坠和鱼钩。
掌握抽动技巧:熟练掌握各种抽动手法，灵活运用，有效刺激深海鱼类的捕食欲望。
注意安全:深海钓鱼需注意安全，穿着救生衣，关注天气和海况变化，确保人身安全。
环保垂钓:不乱扔垃圾，不破坏海洋环境，做到文明垂钓，可持续利用海洋资源。
