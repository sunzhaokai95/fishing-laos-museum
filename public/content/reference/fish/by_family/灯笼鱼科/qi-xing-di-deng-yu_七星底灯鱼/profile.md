# 七星底灯鱼

- 来源：https://fishweather.cn/fish/qi-xing-di-deng-yu
- 学名：Benthosema pterotum
- 科：灯笼鱼科
- 纲：辐鳍鱼纲
- 目：灯笼鱼目
- 水层：深海中层
- 食性：主要以浮游动物为食，包括桡足类、磷虾等小型甲壳动物，以及其他小型鱼类和发光生物。
- 栖息环境：主要栖息于热带和亚热带海域的深海中层水域，通常在水深200-1000米甚至更深的地方活动。
- 适温：未知，推测为深海低温环境，可能在5-15度之间 °C
