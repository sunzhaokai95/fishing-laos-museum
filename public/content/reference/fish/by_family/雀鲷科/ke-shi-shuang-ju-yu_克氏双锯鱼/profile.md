# 克氏双锯鱼

- 来源：https://fishweather.cn/fish/ke-shi-shuang-ju-yu
- 学名：Amphiprion clarkii
- 科：雀鲷科
- 纲：辐鳍鱼纲
- 目：鲈形目
- 水层：中层,底层
- 食性：杂食性，主要以藻类、小型无脊椎动物（如桡足类、片脚类）和海葵触手上的食物残渣为食。
