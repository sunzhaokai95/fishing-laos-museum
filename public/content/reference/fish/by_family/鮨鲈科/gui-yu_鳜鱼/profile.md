# 鳜鱼

- 来源：https://fishweather.cn/fish/gui-yu
- 学名：Siniperca chuatsi
- 科：鮨鲈科
- 纲：辐鳍鱼纲
- 目：鲈形目
- 水层：中层,底层
- 食性：肉食性，性凶猛，主要捕食其他鱼类和虾类。幼鱼阶段亦摄食小型鱼虾及水生昆虫。
- 栖息环境：喜栖息于水质清澈、水流缓慢或静止的水域，常隐匿于水草丛生、石砾或树枝杂乱处。常见于湖泊、水库、江河缓流段及池塘等水体。
- 适温：20-30 °C

基础档案
体侧扁，背部隆起，呈纺锤形。口大，端位，下颌略突出，上下颌及腭骨均具锐齿。眼较小，位于头侧上方。背鳍连续，略呈弧形，硬棘部与软鳍部相连。尾鳍圆形。体色为黄褐色或棕黄色，体侧散布不规则黑色或棕褐色斑纹，腹部灰白色。

路亚钓法
路亚鳜鱼的关键在于搜索和泳层控制。采用抛投-收线的循环方式，大范围搜索标点。根据水深和鳜鱼的活动水层，选择合适的拟饵和操控手法，控制拟饵的泳层。可采用匀速收回、抽停结合、拖拽等手法，模仿小鱼小虾的游动姿态，诱惑鳜鱼攻击。
矶钓/船钓/筏钓
矶钓、船钓、筏钓鳜鱼多采用活饵或冰鲜饵。选择水流较缓、水底结构复杂的区域作为钓点。将饵料抛投至标点，待鱼咬钩后及时扬竿。可采用定点守钓或拖钓的方式。

实战技巧
搜索钓点:鳜鱼常伏击捕食，选择水草、乱石、树桩等障碍物附近作为重点搜索区域。
泳层探测:鳜鱼活动泳层随季节、天气变化，尝试不同泳层，找到鳜鱼所在水层。
标点精细化:在确定大致标点后，可进行精细化搜索，如障碍物边缘、深浅水交界处等。
逗钓结合:路亚过程中，可适当加入逗钓动作，如抽停、拖拽等，增加拟饵的动态效果。
根据鱼情调整:根据鳜鱼的活性和觅食欲望，调整拟饵类型、颜色、操控手法和搜索速度。
