# 尖吻鲈

- 来源：https://fishweather.cn/fish/jian-wen-lu
- 学名：Lates calcarifer
- 科：尖吻鲈科
- 纲：辐鳍鱼纲
- 目：鲈形目
- 水层：中上层
- 食性：肉食性，幼鱼主要摄食小型甲壳类、枝角类及水生昆虫等，成鱼则捕食鱼类、虾、蟹等。
- 栖息环境：广盐性鱼类，可生活于淡水、半咸水及海水中。幼鱼多栖息于红树林、河口、浅滩等近岸水域，成鱼则分布较广，可进入河川下游及沿岸
- 适温：24-32 °C

基础档案
体延长而侧扁，背腹缘均凸出，呈纺锤形。头中大，前端尖细。吻尖。口大，前位，倾斜。下颌稍突出，颌骨、犁骨及腭骨均具绒毛状齿。前鳃盖骨后缘具锯齿，隅角及下缘具少数犬齿。鳃盖骨后缘具一短棘。鳞中大，侧线完全。背鳍两个，彼此分离；第一背鳍硬棘8根，第二背鳍鳍条1鳍棘，10-11鳍条；臀鳍3鳍棘，7-8鳍条；尾鳍圆形。体背部青灰色或棕褐色，腹部银白色。幼鱼体侧具若干黑色横带，成鱼则消失。

矶钓
矶钓尖吻鲈主要在礁石区或防波堤等海域进行。钓组可选择半游动或全游动，根据水流情况调整。饵料以海মাছ或虾为主，也可使用拟饵。作钓时注意安全，穿防滑鞋，注意海浪。
路亚钓
路亚钓尖吻鲈主要采用搜索钓法，在标点区域内不断抛投、收线，搜索目标鱼。操控手法上，可采用匀速收线、抽停结合、跳底等多种方式，模仿小鱼小虾的游动姿态，吸引尖吻鲈攻击。注意观察水面情况，如有炸水、追逐等现象，及时抛投。

实战技巧
精准抛投:尖吻鲈常藏匿于障碍物附近，精准抛投至标点能提高中鱼率。
泳层搜索:尖吻鲈活动泳层较广，需根据水深和鱼情变化，尝试不同泳层搜索。
障碍区作钓技巧:在障碍区作钓时，注意控鱼力度，避免挂底跑鱼。
逗钓结合:在标点反复逗钓，增加路亚饵在目标鱼视野中停留时间，提高攻击机会。
夜钓技巧:夜钓时，可选择夜光路亚饵或搭配夜光棒，提高诱鱼效果。
