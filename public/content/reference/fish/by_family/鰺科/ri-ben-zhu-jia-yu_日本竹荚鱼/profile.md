# 日本竹荚鱼

- 来源：https://fishweather.cn/fish/ri-ben-zhu-jia-yu
- 学名：Trachurus japonicus
- 科：鰺科
- 纲：辐鳍鱼纲
- 目：鲈形目
- 水层：上层,中上层
- 食性：肉食性，主要摄食小型鱼类、甲壳类、头足类及浮游动物等。
- 栖息环境：近海暖温性中上层鱼类，主要栖息于近岸沿海及岛屿周围海域，也进入港湾和河口区。喜集群活动。
- 适温：15-25 °C

基础档案
体延长，侧扁，呈纺锤形。背腹缘弧形，背部稍隆起。头中等大，稍侧扁。吻尖，口大，前位，倾斜。上下颌及犁骨、腭骨均具细齿。眼大，具发达的脂眼睑。鳃孔大，鳃膜分离。体被小圆鳞，易脱落；侧线完全，侧线鳞具棱棘，形成明显的侧线骨板。背鳍2个，连续，第一背鳍鳍棘较弱，第二背鳍与臀鳍相对。尾鳍深叉形。体背部呈蓝绿色或灰蓝色，腹部银白色。体侧中央具一条金黄色纵带，幼鱼期明显。

怎么钓
路亚钓
路亚钓竹荚鱼主要采用搜索钓法，通过快速抛投和回收拟饵，搜索鱼群。操作手法上，可采用快速匀速回收、抽停结合、跳底等方式，模拟小鱼或虾的游动姿态，吸引竹荚鱼攻击。需注意观察水面情况，如发现海鸟聚集或水面有炸水现象，很可能是竹荚鱼群活动的迹象。
船钓
船钓竹荚鱼可采用沉底钓或胴突钓法。沉底钓使用铅坠将钓组沉至海底，通过竿尖的动作判断鱼讯。胴突钓则使用多钩串钩，挂虾肉或鱼肉等饵料，在不同水层搜索鱼群。船钓时需根据潮汐和水流情况调整钓位和钓法。

实战技巧
搜索鱼群:利用探鱼器或观察海面鸟类活动，快速找到竹荚鱼群。
掌握泳层:竹荚鱼活动水层不固定，需通过不断变换拟饵和钓法，测试不同水层。
灵活变换手法:根据鱼情调整拟饵的泳姿和操作手法，如快慢结合、抽停结合等。
注意潮汐:涨潮和退潮时，水流变化会影响鱼的觅食，需根据潮汐变化调整钓位和钓法。
使用鱼轮泄力:竹荚鱼拉力较强，使用带有泄力系统的鱼轮，防止断线跑鱼。
推荐钓点类型
近海、岛屿周围、港口、防波堤、礁石区
