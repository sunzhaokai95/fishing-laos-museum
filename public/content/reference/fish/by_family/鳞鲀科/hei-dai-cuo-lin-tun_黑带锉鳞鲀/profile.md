# 黑带锉鳞鲀

- 来源：https://fishweather.cn/fish/hei-dai-cuo-lin-tun
- 学名：Rhinecanthus rectangulus
- 科：鳞鲀科
- 纲：辐鳍鱼纲
- 目：鲀形目
- 水层：底层,中层
- 食性：杂食性，主要以藻类、小型无脊椎动物（如甲壳类、软体动物）、以及少量的珊瑚和鱼卵为食。
- 栖息环境：主要栖息于热带珊瑚礁和岩礁区域，也可见于泻湖和海草床附近。
- 适温：24-30 °C

基础档案
体呈卵圆形，侧扁，背腹缘皆为弧形。头部轮廓陡峭，吻部突出，口小且端位。眼睛小，位于头侧上方。体表粗糙，鳞片特化为盾状鳞。体色鲜艳，底色为浅黄色至灰白色，体侧具有多条黑色斜带，延伸至腹部，背鳍和臀鳍黄色，尾鳍基部黑色，末端白色。第一背鳍棘强大且可锁定，第二背鳍和臀鳍相对。

怎么钓
矶钓
矶钓可采用底钓或浮游矶钓的方式。底钓时使用重铅将钓组沉至海底，诱饵可以使用海胆、贝类等；浮游矶钓则利用潮汐和海流，将饵料带到更广阔的区域，寻找鱼踪。
路亚海钓
路亚钓法主要针对斜带吻棘鲀的捕食习性，通过操控路亚模拟其猎物，吸引攻击。在珊瑚礁或岩礁区搜索，可采用快速或慢速回收结合抽动、跳动等手法，使路亚在水中呈现不规则运动轨迹，诱发鱼的攻击。

实战技巧
选择合适的钓点:斜带吻棘鲀常在珊瑚礁和岩礁附近活动，选择这些区域作为钓点。
使用亮色路亚:斜带吻棘鲀对颜色鲜艳的路亚反应较好，可选择红色、黄色、橙色等亮色路亚。
注意操控手法:根据不同的路亚类型和鱼情，灵活变换抽动、跳动、匀速回收等手法，提高诱鱼效果。
控制作钓节奏:斜带吻棘鲀咬口凶猛，但有时也较为谨慎，需要根据实际情况调整作钓节奏。
做好安全防护:在礁石区作钓时，注意防滑，穿戴防滑鞋和救生衣，确保安全。
