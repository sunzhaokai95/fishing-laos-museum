# 细条天竺鲷

- 来源：https://fishweather.cn/fish/xi-tiao-tian-zhu-diao
- 学名：Apogon lineatus
- 科：天竺鲷科
- 纲：辐鳍鱼纲
- 目：鲈形目
- 水层：底层
- 食性：肉食性，主要以小型底栖无脊椎动物为食，如甲壳类、多毛类和小鱼。
- 栖息环境：主要栖息于珊瑚礁和岩礁区域的浅水区，也可见于海草床和红树林边缘。
- 适温：24-30 °C

怎么钓
微型路亚
微型路亚钓法针对小型鱼种，需要使用轻巧的装备和微小的拟饵。搜索礁石缝隙、水草区域等，采用慢速拖动或跳底的方式，吸引条纹天竺鲷的注意。注意拟饵的泳姿和颜色搭配。
矶钓
矶钓条纹天竺鲷时，钓组宜轻巧灵敏，钓位选择在礁石缝隙或近岸浅水区域。作钓时可采用逗钓方式，轻微晃动饵料吸引鱼的注意。注意观察浮漂的细微动作，及时提竿。

实战技巧
选择合适的钓点，条纹天竺鲷常在礁石缝隙、水草丛生处或阴暗处活动。
使用轻巧灵敏的钓具，提高鱼讯的感知度。
饵料要小而精，符合条纹天竺鲷的口型。
采用逗钓或慢速拖动的方式，增加诱鱼效果。
耐心等待，仔细观察浮漂或竿梢的细微动作。
