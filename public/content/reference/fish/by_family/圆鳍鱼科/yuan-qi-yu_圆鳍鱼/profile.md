# 圆鳍鱼

- 来源：https://fishweather.cn/fish/yuan-qi-yu
- 学名：Cyclopterus lumpus
- 科：圆鳍鱼科
- 纲：辐鳍鱼纲
- 目：鲉形目
- 水层：底层,近底层
- 食性：主要以甲壳类动物、小型鱼类、无脊椎动物、藻类和水母等为食。
- 栖息环境：寒温带海洋，岩石底部，海藻床，产卵期会在浅水区，幼鱼时期为浮游生活。
- 适温：5-15 °C

## 页面正文

圆鳍鱼
Cyclopterus lumpus
别名：
蛤蟆鱼、狮子鱼
基础档案
身体呈圆球形，皮肤肉质且松软，表面有瘤状突起。腹鳍特化为吸盘，用于吸附在物体表面。体色多变，常见灰色、蓝绿色、棕色或深绿色，可随环境改变。头部较大，眼睛小。
纲
辐鳍鱼纲
目

科
圆鳍鱼科
水层
底层,近底层
食性
主要以甲壳类动物、小型鱼类、无脊椎动物、藻类和水母等为食。
栖息环境
寒温带海洋，岩石底部，海藻床，产卵期会在浅水区，幼鱼时期为浮游生活。
