# 长丝[鱼芒]

- 来源：https://fishweather.cn/fish/zhang-si--yu-mang
- 学名：Pangasius sanitwongsei
- 科：巨鲶科
- 纲：辐鳍鱼纲
- 目：鲇形目
- 水层：底层
- 食性：杂食性，幼鱼主要以小型无脊椎动物、昆虫幼虫等为食，成鱼食性广泛，包括鱼类、甲壳类、水生植物、水果和腐肉等。
- 栖息环境：主要栖息于湄公河干流深水河段，特别是深潭和急流区域，幼鱼有时会在较浅水域活动。
- 适温：24-30 °C

基础档案
体型巨大，呈纺锤形，略侧扁。头部宽大且扁平，吻部钝圆。口下位，口裂宽阔，上下颌具绒毛状齿带。具须2对，颌须长而明显，颏须短小。体表光滑无鳞。背鳍短小，胸鳍和腹鳍中等大小，脂鳍发达，尾鳍深叉形。体色背部呈灰黑色或深灰色，腹部银白色。
