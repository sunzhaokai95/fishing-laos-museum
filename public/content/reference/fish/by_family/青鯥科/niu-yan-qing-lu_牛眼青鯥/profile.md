# 牛眼青鯥

- 来源：https://fishweather.cn/fish/niu-yan-qing-lu
- 学名：Scombrops boops
- 科：青鯥科
- 纲：辐鳍鱼纲
- 目：鲈形目
- 水层：中上层,中层
- 食性：肉食性，主要捕食小型鱼类、甲壳类（如虾、蟹）和头足类（如鱿鱼、章鱼）。幼鱼阶段可能摄食浮游动物。

- 适温：16-28 °C

Scombrops boops
别名：
黑鲳、篮鳍鲳、大眼鲳、膨鱼
基础档案
蓝鳍鲳体延长而侧扁，呈卵圆形。头大，眼大，口大且倾斜，下颌略突出。体被小圆鳞，侧线明显。背鳍两个，分离，第一背鳍棘状，第二背鳍与臀鳍相对且延长。尾鳍深叉形。体背部呈蓝黑色或深灰色，腹部银白色，各鳍为黑色或深灰色，胸鳍和腹鳍基部略带蓝色，这也是其“蓝鳍”名称的由来。幼鱼时期体色较浅。

怎么钓
船钓
船钓蓝鳍鲳主要采用沉底钓或铁板钓法。沉底钓可使用活饵或死饵，将钓组放到海底等待鱼咬钩。铁板钓则使用铁板假饵，通过抽动竿梢，使铁板在水中呈现跳跃的姿态，吸引蓝鳍鲳攻击。需要根据水深和鱼情选择合适的钓点和钓法。
岸矶钓
岸矶钓蓝鳍鲳主要选择礁石区或防波堤等水深较深的位置。可使用阿波钓法或直感钓法，根据潮汐和水流变化调整钓组和钓棚深度。注意安全，选择安全的钓位，避免在风浪大的天气进行矶钓。
路亚海钓
路亚海钓蓝鳍鲳可使用米诺、铅笔、VIB、铁板等多种路亚饵。主要在礁石区、沉船区或鱼群聚集区进行搜索。采用快抽、慢抽、跳底等手法，使路亚饵在水中呈现自然的泳姿，吸引蓝鳍鲳攻击。注意根据水深和鱼情更换不同类型的路亚饵和泳层。

实战技巧
选择合适的钓点:蓝鳍鲳喜欢栖息在礁石区、沉船区等复杂地形，这些地方往往是藏鱼的好位置。
根据潮汐变化调整钓法:涨潮和退潮时，蓝鳍鲳的觅食习性可能发生变化，需要根据潮汐变化调整钓法和钓点。
使用合适的饵料和路亚:根据季节和鱼情选择蓝鳍鲳喜欢的饵料或路亚，提高中鱼率。
注意钓棚深度:蓝鳍鲳通常在中上层或中层活动，需要根据鱼情调整钓棚深度。
掌握正确的抽竿时机:蓝鳍鲳咬钩时，竿梢通常会有明显的下顿或弯曲，需要及时扬竿刺鱼。
