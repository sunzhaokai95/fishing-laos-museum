# 鮸鱼

- 来源：https://fishweather.cn/fish/mian-yu
- 学名：Miichthys miiuy
- 科：石首鱼科
- 纲：辐鳍鱼纲
- 目：鲈形目
- 水层：底层,近底层
- 食性：肉食性，主要捕食小型鱼类、虾、蟹等甲壳类及头足类
- 栖息环境：近海岩礁、砂砾底质海区，也进入河口咸淡水域
- 适温：10-25 °C

基础档案
体延长，侧扁，呈纺锤形。头中等大，稍侧扁。吻尖突。口大，端位，斜裂。上下颌、犁骨及腭骨均具细齿。前鳃盖骨后缘具锯齿，鳃盖骨后缘光滑。眼间隔稍宽，微凹。鳞小，栉鳞，侧线完全。背鳍1个，硬棘部与软条部相连。胸鳍宽大，圆形。腹鳍胸位。臀鳍与背鳍软条部相对。尾鳍截形或稍内凹。体背部及侧面灰黑色或棕黑色，腹部灰白色。各鳍灰黑色。

怎么钓
矶钓
矶钓海鲈鱼的关键在于钓位的选择和诱钓结合。选择岩礁突出、水流湍急、浪花翻滚的区域作为钓点，这些地方溶氧量高，饵料丰富，易聚集海鲈鱼。采用诱饵窝料提前打窝诱鱼，如南极虾砖、海藻等。钓饵可选择活虾、沙蚕、海মাছ等。作钓时，注意观察浮漂动作，及时扬竿刺鱼。矶钓需注意安全，穿着防滑鞋，注意潮汐变化。
路亚钓
路亚海鲈鱼的关键在于搜索和泳姿控制。需要不断抛投、收线，搜索鱼群所在水层和区域。根据标点和鱼情选择合适的泳层和搜索速度。操控手法上，可以通过匀速收线、抽停结合、跳底等手法，使路亚呈现不同的泳姿，吸引海鲈鱼攻击。注意观察竿梢和线组的细微变化，及时扬竿刺鱼。

实战技巧
根据潮汐选择钓点和钓时，涨潮和退潮时，海鲈鱼活动频繁
根据水色选择路亚颜色和泳层，浑水用亮色系路亚，清水用自然色系路亚
掌握海鲈鱼的习性，多在水流交汇处、障碍物附近搜索
灵活运用不同的路亚手法，刺激海鲈鱼攻击
注意安全，海边垂钓注意防滑防浪
推荐钓点类型
岩礁区、码头、防波堤、海湾、河口咸淡水域
