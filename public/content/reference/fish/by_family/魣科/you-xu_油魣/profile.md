# 油魣

- 来源：https://fishweather.cn/fish/you-xu
- 学名：Sphyraena pinguis
- 科：魣科
- 纲：辐鳍鱼纲
- 目：鲈形目
- 水层：中上层
- 食性：肉食性，主要捕食小型鱼类、甲壳类和头足类等。
- 栖息环境：主要栖息于热带和亚热带沿岸岩礁、珊瑚礁或开阔海域，也进入河口和海湾。
- 适温：20-30 °C

## 页面正文

油魣
Sphyraena pinguis
别名：
竹梭、䲃鱼
基础档案
体延长，亚圆筒形，稍侧扁。头长而尖，吻尖突。口大，口裂深，达眼后缘下方。上下颌具强犬齿，腭骨和犁骨具绒毛状齿。前鳃盖骨后缘圆，下缘平直。体被小圆鳞，侧线完全。背鳍两个，第一背鳍位于腹鳍起点上方，第二背鳍与臀鳍相对。体背侧灰褐色或青灰色，腹侧银白色。各鳍淡色。
纲
辐鳍鱼纲
目
鲈形目
科

水层
中上层
食性
肉食性，主要捕食小型鱼类、甲壳类和头足类等。

主要栖息于热带和亚热带沿岸岩礁、珊瑚礁或开阔海域，也进入河口和海湾。

实战技巧
根据水深和鱼层选择合适的拟饵和钓法。
掌握抽动节奏和幅度，模仿目标鱼的捕食对象。
注意观察水面情况，寻找鱼群活动的迹象。
使用前导线，防止被牙齿咬断主线。
中鱼后及时泄力，避免断线跑鱼。
