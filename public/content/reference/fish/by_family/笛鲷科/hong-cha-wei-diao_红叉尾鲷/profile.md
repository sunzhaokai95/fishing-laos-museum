# 红叉尾鲷

- 来源：https://fishweather.cn/fish/hong-cha-wei-diao
- 学名：Aphareus rutilans
- 科：笛鲷科
- 纲：辐鳍鱼纲
- 目：鲈形目
- 水层：中层,底层
- 食性：肉食性，主要以小型鱼类、甲壳类（如虾、蟹）和头足类（如鱿鱼、章鱼）为食。幼鱼可能摄食浮游动物。

- 适温：22-30

基础档案
体延长，略侧扁，呈纺锤形。头中等大小，吻尖。口大，端位，上下颌均具细齿，锄骨和腭骨无齿。眼大，眶前骨窄。前鳃盖骨后缘具锯齿，鳃盖骨具弱棘。体被中等大小栉鳞，侧线完全。背鳍和臀鳍具鞘鳞。背鳍连续，单一，硬棘部和软条部之间无明显凹刻，硬棘X，软条10-11。臀鳍小，硬棘III，软条8。胸鳍尖长。尾鳍深叉形。体背部及侧上方呈鲜艳的红色或橘红色，腹部颜色较浅，偏银色或粉红色。幼鱼体色较浅，随着成长颜色逐渐加深。

怎么钓
路亚钓
路亚钓小齿若花鲈的关键在于选择合适的拟饵和操控手法。常在礁石区、珊瑚礁附近或水流湍急处搜索。可采用抽停结合、快速收线等手法，模仿小鱼或受伤鱼的泳姿来吸引若花鲈攻击。注意探查不同水层，若花鲈可能在水面、中层或底层活动。
船钓拖钓
拖钓主要在开阔水域或深海区进行。选择合适的拖钓速度和航线，将拟饵或钓饵拖曳在若花鲈可能出没的水层。注意调整拖钓深度和距离，根据鱼情变化调整拖钓策略。中鱼后需及时收线，与鱼展开力量较量。
铁板钓
铁板钓法主要针对深水区或离岸较远的礁石区。通过操控铁板在水中上下跳动、快速抽动等方式，模拟猎物吸引若花鲈。注意控制铁板下沉速度和深度，根据水流和鱼情调整抽动频率和幅度。中鱼后需迅速扬竿，利用竿子的腰力与鱼搏斗。

实战技巧
精准探点:利用探鱼器或经验判断，找到若花鲈藏匿的礁石区、暗礁或水下结构。
泳层搜索:若花鲈可能在不同水层活动，从表层到深层逐层搜索，调整拟饵和钓法。
拟饵选择多样化:准备不同类型、颜色和泳姿的拟饵，根据鱼情和水色灵活更换。
搏鱼技巧:若花鲈力量较大，中鱼后需稳健搏鱼，利用泄力卸力，避免硬碰硬断线跑鱼。
安全第一:海钓注意安全，穿着救生衣，关注天气和海况变化，避免恶劣天气出海。
