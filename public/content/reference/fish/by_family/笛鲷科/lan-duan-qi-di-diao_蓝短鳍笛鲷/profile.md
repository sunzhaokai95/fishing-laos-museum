# 蓝短鳍笛鲷

- 来源：https://fishweather.cn/fish/lan-duan-qi-di-diao
- 学名：Aprion virescens
- 科：笛鲷科
- 纲：辐鳍鱼纲
- 目：鲈形目
- 水层：中上层,中层
- 食性：肉食性，主要捕食鱼类、甲壳类（如虾、蟹）和头足类等海洋无脊椎动物。
- 栖息环境：主要栖息于热带和亚热带的珊瑚礁区和岩礁海域，也常见于开阔水域和近岸浅水区。幼鱼常在沿岸海草床或红树林中活动。
- 适温：22-28 °C

基础档案
体延长，略侧扁，呈纺锤形。头中大，前端尖细。吻尖。口大，端位，口裂斜。上下颌牙细尖，颌齿呈绒毛状。前鳃盖骨后缘具锯齿。鳃耙中等长。体被栉鳞，侧线完全，侧线上鳞列数47-52。背鳍连续，单一，鳍棘部与鳍条部相连，无明显缺刻，鳍棘Ⅹ，鳍条11-12。臀鳍鳍棘Ⅲ，鳍条8。尾鳍叉形。体背部及侧面呈蓝绿色或灰绿色，腹部银白色。各鳍均为淡黄绿色或黄绿色，鳍边颜色较浅。

怎么钓
船钓
船钓绿鳍鲷主要采用沉底钓法或铁板钓法。沉底钓法使用重铅将钓组沉至海底，以鱼块、鱿鱼等为饵料，等待鱼咬钩。铁板钓法使用金属Jig在水中进行快速抽动，吸引绿鳍鲷攻击。船钓时需注意选择合适的钓点，如礁石区、沉船区或鱼群聚集区。
路亚海钓
路亚海钓绿鳍鲷主要采用抛投搜索钓法和垂直Jigging钓法。抛投搜索钓法适用于近岸礁石区和浅海区域，通过不断抛投和回收拟饵，搜索目标鱼。垂直Jigging钓法适用于船钓或深水区，通过操控金属Jig或软饵在水中进行上下抽动，吸引绿鳍鲷攻击。

实战技巧
选择合适的钓点:绿鳍鲷常栖息于礁石区和沉船区，这些区域是作钓的重点。
掌握潮汐规律:涨潮和退潮时，水流变化会带来更多食物，是钓鱼的好时机。
灵活运用拟饵和钓法:根据鱼情和钓场环境选择合适的拟饵和钓法，如水清时可使用颜色鲜艳的软饵或亮片，水浑时可使用颜色深沉的金属Jig或VIB。
注意探底和搜索:绿鳍鲷常在水底活动，需注意探底，并进行大范围搜索，寻找鱼群。
中鱼后及时扬竿和控鱼:绿鳍鲷咬口凶猛，中鱼后需及时扬竿，并迅速控鱼，防止鱼钻入礁石或逃脱。
