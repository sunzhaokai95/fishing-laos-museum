# 白斑笛鲷

- 来源：https://fishweather.cn/fish/bai-ban-di-diao
- 学名：Lutjanus bohar
- 科：笛鲷科
- 纲：辐鳍鱼纲
- 目：鲈形目
- 水层：中层,底层
- 食性：肉食性，主要以鱼类、甲壳类（如虾、蟹）和头足类（如鱿鱼、章鱼）为食。幼鱼阶段可能摄食小型浮游动物。
- 栖息环境：主要栖息于热带和亚热带珊瑚礁区及其附近海域，也见于岩礁、潟湖和外海斜坡。幼鱼有时会出现在河口和红树林。
- 适温：22-28 °C

基础档案
体呈椭圆形，侧扁而高，背缘隆起。头中大，吻尖。口大，端位，上下颌牙细尖。前鳃盖骨后缘具锯齿，鳃盖骨具扁棘。体被中大栉鳞，侧线完全。背鳍连续，硬棘部与软条部相连，无明显缺刻。体背部及侧上部为红色或橙红色，腹部颜色较浅，幼鱼体侧具数条蓝色纵带，成鱼纵带消失。各鳍红色或橙红色。

路亚海钓
路亚海钓红鲉主要采用底钓和跳底钓法。将拟饵抛投至标点区域，让其沉至海底，然后通过抽动竿梢或匀速收线，使拟饵在水底或中下层呈现跳跃、拖动等动态，模拟小鱼小虾等猎物，吸引红鲉攻击。需要根据水流、深度和鱼情调整拟饵的泳层和动态。

实战技巧
选择合适的拟饵:根据水深、水流和目标鱼大小选择合适的拟饵类型、颜色和尺寸。
探明鱼层:红鲉可能在水底或中下层活动，通过不同泳层的搜索，快速找到鱼层。
灵活运用抽动手法:根据鱼情调整抽动频率和幅度，尝试不同的手法刺激鱼攻击。
注意障碍物:珊瑚礁区和岩礁区障碍物较多，抛投和操控拟饵时注意避开挂底。
使用前导线:在复杂水底环境中使用耐磨损的碳氟前导线或钢丝前导线，防止切线。
