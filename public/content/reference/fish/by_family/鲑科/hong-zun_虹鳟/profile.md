# 虹鳟

- 来源：https://fishweather.cn/fish/hong-zun
- 学名：Oncorhynchus mykiss
- 科：鲑科
- 纲：辐鳍鱼纲
- 目：鲑形目
- 水层：中上层
- 食性：肉食性鱼类，主要捕食水生昆虫、甲壳类、软体动物、小型鱼类等。人工养殖条件下，可投喂配合饲料。
- 栖息环境：喜栖息于清澈、低温、溶氧量高的淡水水域，如溪流、河流、湖泊等。对水质要求较高，是水质清洁的指示性鱼类。
- 适温：10-20 °C

别名：
美国虹鳟、赤鳟、瀑布鱼、虹鳟鲑
基础档案
体呈纺锤形，略侧扁，体色鲜艳，成鱼体侧有一条明显的粉红色或彩虹色纵带，故名“虹鳟”。全身遍布小黑斑，以背部和脂鳍上最多。头部较小，吻尖，口端位，上下颌均有细齿。脂鳍发达，尾鳍叉形。鳞片细小，排列紧密。

食性

栖息环境
喜栖息于清澈、低温、溶氧量高的淡水水域，如溪流、河流、湖泊等。对水质要求较高，是水质清洁的指示性鱼类。

虹鳟
怎么钓
飞蝇钓
飞蝇钓虹鳟鱼的关键在于拟饵的制作和自然的呈现。根据水生昆虫的种类和羽化情况，选择或自制合适的飞蝇拟饵。抛投时力求轻柔、精准，将飞蝇自然地漂浮或沉入目标水域。通过控线和mend线等技巧，控制飞蝇的漂流速度和方向，使其呈现自然的泳姿，诱骗虹鳟鱼上钩。注意观察水面是否有鱼的活动迹象，根据鱼情变化调整飞蝇类型和钓法。
路亚钓
路亚虹鳟鱼的关键在于搜索和泳姿控制。根据水域环境和鱼情选择合适的泳层和路亚类型。抛投后，通过匀速收线、抽停结合、快慢结合等手法，赋予路亚生动的泳姿，模拟小鱼小虾等猎物，吸引虹鳟鱼攻击。注意观察水面情况，寻找鱼星或捕食迹象，精准抛投至标点。勤抛竿、多搜索，不放过任何可疑区域。
亮片钓
亮片钓虹鳟鱼的关键在于亮片的选择和泳速的控制。根据水色和光照条件选择合适的亮片颜色和类型，如银色、金色、彩色亮片等。抛投后，控制亮片的泳速，可匀速收线，也可快慢结合，模仿小鱼的游动姿态，吸引虹鳟鱼追逐攻击。在收线过程中，可轻微抽动竿梢，增加亮片的泳姿变化，提高诱鱼效果。注意观察水面情况，寻找鱼星或捕食迹象，精准抛投至标点。

实战技巧
根据水温和水深选择合适的泳层和路亚类型，虹鳟鱼在不同季节和水温条件下，活动泳层会发生变化。
观察水面情况，寻找鱼星、捕食迹象或水流变化，精准抛投至标点。
灵活变换抽动手法和收线速度，尝试不同的泳姿，找到最有效的诱鱼方式。
使用偏光镜，减少水面反光，观察水下情况，更容易发现虹鳟鱼。
勤换钓位，虹鳟鱼活动范围较大，多尝试不同的钓点，增加中鱼机会。
推荐钓点类型
溪流、河流、湖泊、水库、虹鳟鱼养殖场、允许垂钓的封闭水域
